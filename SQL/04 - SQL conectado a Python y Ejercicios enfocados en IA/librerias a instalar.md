# SQL + PYTHON + IA - SANTIAGO BELTRAN
## Librerias para el ejercicio:

---

- Para conectar esa base de datos mySQL primero tenemos que instalar en la terminal:

1. El conector de MySQL para python: 
'''
pip install mysql-connector-python
'''

---

2. Librerías que manejen datos (Pandas-Numpy): 
'''
pip install pandas numpy
'''

---

3. Librerías para los ejercicios de ejemplo con IA: 
- Para regresión y clasificación:
'''
pip install scikit-learn 
'''
- Para las redes neuronales e imágenes:
'''
pip install tensorflow
'''

---

#4. Gráficos: pip install matplotlib
- Dashboard: 
'''
pip install plotly
'''

---

#5. Manejador de Imágenes: 
'''
pip install pillow
'''
